## Getting started
1. `npm install`

## Developing
1. `npm run dev`

## Code Coverage
1. `npm run coverage`

## Code Linting
1. `npm run lint`

## Docker

### Prerequisites
1. Make sure you have the latest Docker Toolbox installed.
1. Make sure you have ran `npm install`

### Dockerizing
1. `npm run docker`
1. `open http://$(docker-machine ip <YOUR_DOCKER_MACHINE_NAME>)`
1. You should see `Hello world!`
